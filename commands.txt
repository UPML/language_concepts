одна строка - [00 00 00 00 00] имеет такой размер
Список команд с их адресами
под адрес команды выделено первые две ячейки
[01] ip x- перейти на строку с номером x
[02] inp x - считать число с клавиатуры записать по адресу х
[03] dec a - из числа по адресу а - вычесть 1
[04] push a - положить на вершину стеку значение по адресу а
[05] pushaddr a - положить на вершину стека адрес а
[06] pop a - снять с вершины стека значение, записать его в а
[07] cmp a b - сравнить a и b по значению
[08] cjump х - если пред сравнение true то прыгнуть на х иначе ничего не делать
[09] add a b - сложить значения а и b результат записать в а
[0a] move x - перейти по адресу х
[0b] ret - снять с вершины стека значение и перейти в ячейку с таким адресом
[0c] copy a b - записать по адресу а значение из b
[0d] out x - вывести значение по адресу х на экран
[0e] stop - завершить исполнение работы
[0f] res x - зарезервировать на стеке место для х переменных
[10] sin синхронизировать окно на стеке и окно вначале программы
[11] print "hey" - вывести на дисплей hey
